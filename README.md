# ChefApp - RecetApp

Aplicación para guardar recetas de cocina con un cuaderno de dibujo integrado para hacer esquemas técnicos y anotaciones visuales.

## Qué hace

- Guarda recetas con ingredientes, cantidades, tiempos y porciones
- Organiza por categorías (Postres, Salado, Bebidas, Saludable)
- Dibuja esquemas técnicos en una pizarra táctil
- Filtra y busca recetas
- Modo oscuro

## Tecnologías

React, Firebase, Vite, Framer Motion

## Instalación

```bash
npm install
npm run dev
```
